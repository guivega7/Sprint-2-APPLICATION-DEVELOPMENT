# Verde Radar

Dashboard para monitoramento remoto de áreas de vegetação com dados fictícios de sensoriamento, pontos geográficos obtidos via OpenStreetMap/Nominatim e clima consultado na Open-Meteo.

## Como executar

1. Instale as dependências com `npm install`.
2. Inicie o projeto com `npm run dev`.
3. Abra a URL exibida pelo Vite no navegador.

## O que o dashboard entrega

- Pelo menos 10 pontos geográficos com latitude e longitude.
- Mapa interativo com marcadores.
- Resumo climático por ponto.
- Leituras fictícias de sensoriamento para compor o risco operacional.

## Arquivo de integrantes

Preencha o arquivo `integrantes.txt` com os nomes completos e os RMs antes da entrega final.
